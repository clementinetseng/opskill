# 後台系統操作指南

## 登入後台

1. 開啟瀏覽器，前往 `https://admin.yourplatform.com`
2. 輸入你的員工帳號與密碼
3. 完成 2FA 驗證 (Google Authenticator)

## 快捷鍵

- `Ctrl + K`: 快速搜尋會員
- `Ctrl + Shift + T`: 開啟交易記錄
- `Ctrl + L`: 登出系統
